### Added
- Search skills from skills.sh with real-time debounced search

- Open skills on skills.sh website
- Open skill repositories on GitHub
- View install counts with formatted numbers (4.8K, 1.2M)
- Client-side filtering for trending skills
- Retry actions for failed API requests
